Complex Contrast Training (CCT) Workout - 1

Main Complex Sets (2 sets each, 4 min rest between sets and blocks)

Session Summary

  • Type: Power
  • Focus: Shoulders / Rotation / Boxing
  • Sets: 2 per block
  • Target Duration: ~45–55 minutes

Warm-Up Guidelines

Warm-up as needed. Rest ~60–90 seconds between warm-up sets.

Barbell / Heavy Lifts:

  • ~30% 1RM x 5 reps
  • ~60–65% 1RM x 3 reps
  • ~75–80% 1RM x 2 reps

Explosive / Rotational Movements:

  • 2–3 progressive sets (build to working intensity)

1. Shoulders/Vertical Push

B1: Push Press or Dumbbell Arnold Press - 3-5 reps

Rest: ~60 seconds, then perform B2.

B2: Overhead Medicine Ball Slams or Shot Put Throws - 6-8 reps

Rest: ~4 minutes, then repeat for set 2.

Rest / Setup: ~3–5 minutes

2. Rotational Power

D1: Landmine Twist (Heavy) - 6 reps per side

Rest: ~60 seconds, then perform D2.

D2: Rotational Medicine Ball Slam/Throw - 6 reps per side

Rest: ~4 minutes, then repeat for set 2.

Rest / Setup: ~3–5 minutes

3. Boxing-Specific Power

F1: Landmine Jab (Heavy) - 6 reps per side

Rest: ~60 seconds, then perform F2.

F2: Plyometric Uppercut Throws with Medicine Ball - 6 reps per side

Rest: ~4 minutes, then repeat for set 2.
