Complex Contrast Training (CCT) Workout - 2

Main Complex Sets (2 sets each, 4 min rest between sets and blocks)

Session Summary

  • Type: Power
  • Focus: Chest / Back / Legs
  • Sets: 2 per block
  • Target Duration: ~55–70 minutes

Warm-Up Guidelines

Warm-up as needed. Rest ~60–90 seconds between warm-up sets.

Barbell / Heavy Lifts:

  • ~30% 1RM x 5 reps
  • ~60–65% 1RM x 3 reps
  • ~75–80% 1RM x 2 reps

Explosive / Rotational Movements:

  • 2–3 progressive sets (build to working intensity)

1. Chest/Push Focus

A1: Barbell Bench Press - 3-5 reps @ 85-90% 1RM

Rest: ~60 seconds, then perform A2.

A2: Plyometric Push-Ups or Clap Push-Ups - 5-8 reps

Rest: ~4 minutes, then repeat for set 2.

Rest / Setup: ~3–5 minutes

2. Back/Pull Focus

C1: Weighted Pull-Ups or Barbell Rows - 4-6 reps

Rest: ~60 seconds, then perform C2.

C2: Band-Resisted Row Explosions or Explosive Renegade Rows - 6-8 reps

Rest: ~4 minutes, then repeat for set 2.

Rest / Setup: ~3–5 minutes

3. Lower Body/Leg Power

Warm-Up (Back Squat):

  • ~30% 1RM x 5 reps
  • ~60-65% 1RM x 3 reps
  • ~75-80% 1RM x 2 reps

E1: Barbell Back Squat - 3-5 reps @ 80-85% 1RM

Rest: ~60 seconds, then perform E2.

E2: Box Jumps or Depth Jumps - 5-8 reps

Rest: ~4 minutes, then repeat for set 2.
